# 📉 Telco Customer Churn Prediction: An Evolutionary Approach

This project demonstrates a full Data Science lifecycle, moving from a simple baseline to a high-performance ensemble model. 

### 📂 Project Structure
* **[v1-Baseline](./v1-Baseline):** Exploratory Data Analysis and initial Logistic Regression.
* **[v3-Final](./v3-Final):** Production-ready model using SMOTETomek and Ensemble Optimization.

### 🚀 The Goal
To identify customers likely to cancel their service (churn) so the marketing team can intervene with retention offers.

### 🛠️ Key Technologies
* **Python** (Pandas, Scikit-Learn, XGBoost)
* **Imbalanced-Learn** (SMOTE for handling lopsided data)
* **Version Control** (GitHub-based model iteration)
