# 📁 v1: Baseline Churn Model
This folder contains the initial exploratory phase of the project.

### 🎯 Goal
Establish a starting point for performance using raw data and a standard Logistic Regression model.

### 🔍 Key Features
* **Exploratory Data Analysis (EDA):** Initial look at customer behavior.
* **Simple Preprocessing:** Basic categorical encoding.
* **Baseline Model:** Logistic Regression (no tuning).

### 📈 Results
* **Focus:** Establishing initial metrics to compare against future optimizations.
* **Next Step:** Moving to **v2** to implement Feature Scaling and Hyperparameter Tuning.
