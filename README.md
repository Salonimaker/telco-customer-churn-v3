# 📉 Telco Customer Churn Prediction: An Evolutionary Approach

This project demonstrates a full Data Science lifecycle, moving from a simple baseline to a high-performance ensemble model through iterative optimization.

---

## 📂 Project Structure

* **v1-Baseline:** Initial Data Cleaning and Logistic Regression to establish a performance floor.
* **v2-Ensemble:** Transitioned to **Random Forest** with **Custom Threshold Tuning (0.22)** to prioritize high-sensitivity (**83.6% Recall**).
* **v3-Final:** Production-ready architecture using **SMOTETomek** for data balancing and Ensemble Optimization.

---

## 🚀 The Evolution of Churn Detection

| Version | Model Type | Strategy | Key Result |
| :--- | :--- | :--- | :--- |
| **v1** | Logistic Regression | Baseline Analysis | Established starting metrics. |
| **v2** | **Random Forest** | **Recall-First Tuning** | **83.6% Recall** (Captured 312/373 churners). |
| **v3** | Optimized Ensemble | Precision/Recall Balance | Maximized F1-Score and business ROI. |
